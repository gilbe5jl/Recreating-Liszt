Project Background

Recreating Liszt is a project that's focused on using machine learning to create music that is similar in style to world-renowned composer and pianist Franz Liszt. Our project is a development team consisting of 7 people, split into two halves:

Web Application

One half (3/7) of the group is focused on the web-application. The members of this team include Hailey Werth, Roger Mawby, and Zachary McDonald. The web app is built from Django for the back-end, and Django Templates using HTML, CSS, and Bootstrap for the front-end. The web-app features a fully functioning user-registration system, login/logout system, and a clear, simple UI to navigate to all of our pages. While rummaging through the site you may find: a page containing a Python notebook so that you can view our code, links to Google Colab and our Github repository so that you can run/edit the code yourself, a small biography about Franz Liszt, and a small biography about our dev team.

Machine Learning

The other half of the group (4/7) is focused on the machine learning portion of the project. The members of that team include Zack Saffron, Brady Granquist, Hunter Francek, and Jalen Gilbert. We chose to use Google Colab and and Magenta to accomplish our task of building the AI. Magenta is a Python library that's powered by TensorFlow that helps us generate the music. The goal of this project is simple: we aim to use Machine Learning to recreate the music of 19th-century composer and pianist Franz Lizst.
